## üåü What's This Magic?

LaravelProjectContainer adalah solusi **All-in-One** untuk developer Laravel yang ingin:
- ‚ú® Setup development environment dalam **1 command**
- üéØ Manage multiple Laravel projects dalam **1 container**
- üöÄ Deploy ke production dengan **terminal commands**
- üé™ Fokus coding, bukan setup environment!

```bash

git clone https://github.com/denis156/LaravelProjectContainer.git
```

## üöÄ Quick Start Guide

### 1. **Prerequisites** 
```bash
# Yang perlu di-install:
- Docker & Docker Compose üê≥
- Git (untuk cloning projects) üì¶
- Terminal yang support colors üé®
```

### 2. **Installation**
```bash
# Clone the magic! ‚ú®
git clone https://github.com/denis156/LaravelProjectContainer.git
cd LaravelProjectContainer

# Setup terminal scripts permissions üîß
chmod +x setup_terminal.sh
./setup_terminal.sh

# Copy environment file
cp .env.example .env

# Start the container! üöÄ
docker-compose up -d

# Wait for magic to happen... ‚è≥
# Check status
docker-compose ps
```

### 3. **Create Your First Project**
```bash
# Enter the container
docker exec -it laravel_frankenphp bash

# Create new Laravel project (auto-configured!) üéØ
./Terminal/project.sh new my-awesome-app

# Start development environment üî•
./Terminal/dev.sh start

# Open in browser (auto!) üåê
./Terminal/dev.sh open
```

### 4. **Development Workflow**
```bash
# Switch between projects üîÑ
./Terminal/project.sh switch another-app

# Run artisan commands üé™
./Terminal/artisan.sh migrate
./Terminal/artisan.sh make model User -mcr

# Manage composer packages üì¶
./Terminal/composer.sh require laravel/telescope --dev

# Database operations üóÑÔ∏è
./Terminal/database.sh backup my-app
./Terminal/database.sh fresh my-app

# Deploy to production üöÄ
./Terminal/deploy.sh deploy my-app production
```

### 5. **Optional: Setup Global Commands**

Want shorter commands? Setup global aliases:

#### üîß **Manual Path Setup (Temporary):**
```bash
# Add to PATH untuk session ini
export PATH="$(pwd)/bin:$PATH"

# Test
project help
```

#### üöÄ **Permanent Setup (Recommended):**
```bash
# For Zsh users (macOS default)
echo 'export PATH="$(pwd)/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc

# For Bash users  
echo 'export PATH="$(pwd)/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc

# Test
project help
```

#### üéØ **Alternative: Use Direct Path**
Prefer manual commands? No setup needed:

```bash
./Terminal/project.sh help
./Terminal/dev.sh start
./Terminal/artisan.sh migrate
```

> **üí° Note:** Manual commands (./Terminal/script.sh) always work without any setup!
